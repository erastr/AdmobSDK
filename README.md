GoogleMobileAds Version: 11.0.0
ExternalDependencyManager Version: 1.2.187

- Admob tarafında oluşturduğunuz reklamları projenizde kolayca kullanabileceğiniz bir paket.

# Kurulum:
1-) Paketi projenize import edin.
2-) Assets->Google Mobile Ads -> Settings kısmında Admob tarafında oluşturduğunuz appid'lerinizi girin. (Editor üzerinden test ederken gerekmez.)
3-) Admob tarafında oluşturduğunuz gerçek reklam idlerini AdmobController.cs'de gerekli alanlara girin.
4-) AdManager ile reklamlarınızı istediğiniz yerden yükleyin veya gösterin.
