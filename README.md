# Versiyonlar
- GoogleMobileAds Version: 11.0.0
- ExternalDependencyManager Version: 1.2.187

# Özet
- Admob tarafında oluşturduğunuz reklamları projenizde kolayca kullanabileceğiniz bir paket.

# Kurulum:
- Paketi projenize import edin.
- Assets->Google Mobile Ads -> Settings kısmında Admob tarafında oluşturduğunuz appid'lerinizi girin. (Editor üzerinden test ederken gerekmez.)
- Admob tarafında oluşturduğunuz gerçek reklam idlerini AdmobController.cs'de gerekli alanlara girin.
- AdManager ile reklamlarınızı istediğiniz yerden yükleyin veya gösterin.
